# Problema 1:

**Entrada:** um vetor de palavras previamente preenchido.<br>

**Saída:** informar qual é a maior palavra do conjunto e seu tamanho.<br>

Exemplo:<br>

**Entrada:**

```
String [] palavras = {"the", "book", "table "};
```

**Saída:**

```
A maior palavra do conjunto é: table
Tamanho: 5
```
